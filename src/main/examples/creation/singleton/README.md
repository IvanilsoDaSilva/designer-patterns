<h2 align="right">PROBLEMA ❌</h2>

Garantir que uma determinada classe tenha apenas uma instância em toda a sua aplicação.

<h2 align="right">SOLUÇÃO 💡</h2>

O singleton resolve esse problema garantindo que uma classe tenha apenas uma instância e fornecendo um ponto global de acesso a essa instância.
